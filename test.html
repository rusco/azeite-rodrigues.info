Configuration

Defaults to https://api.azeite-rodrigues.info
Your API endpoints are relative to this, e.g. /api/subscribe.
Used for /admin/* calls. Not stored anywhere.

Public Endpoints

Subscribe POST /api/subscribe

Sends JSON: {"email": "...", "name": "..."}

Confirm GET /confirm?token=...

Opens /confirm?token=.... You’ll supply the token from the email you received.

Unsubscribe by Token GET /unsubscribe?token=...

Unsubscribe by Email POST /unsubscribe

JSON body: {"email": "..."} or as application/x-www-form-urlencoded.

Brevo Webhook (test) POST /webhooks/brevo

Header X-Webhook-Token is taken from the Configuration panel.

Admin Endpoints (bottom)

Create Issue POST /admin/issue

Requires Authorization: Bearer <ADMIN_TOKEN>.

Send Issue POST /admin/send

Creates deliveries for all status='confirmed' subscribers.
